## Schedule manager


## Dependencies

```
Docker >= 19.03
Docker-compose >= 1.26
```

## Setup

Clone the project.
```console
$ git clone git@github.com:edumoreira1506/schedule-manager.git
$ cd schedule-manager
```

Build docker image:
```console
$ docker build .
```

Up api container image:
```console
$ docker-compose up api
```
